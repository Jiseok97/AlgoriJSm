# 1436 → 영화감독 숌

### 내 코드👨🏻‍💻
```python
n = int(input())
cnt = 0
value = 666

while True:
    if '666' in str(value):
        cnt += 1
    
    if cnt == n:
        print(value); break
    
    value += 1
```

</br>

### 내 생각 📝
- 문제 이해가 좀처럼 쉽지 않아 다른 블로그 글들을 참고하게 되었다.
- 다소 이해가 어려웠지만 구현에는 큰 어려움이 없었던 문제였다.
